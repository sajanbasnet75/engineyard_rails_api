Create new customer API

Creates a new room for the hotel successfully

POST /api/v1/rooms

Parameters

Name Description
hotel_id Hotel id for which the room is added
name name of room
description Description of room
room_type can be Single Double Deluxe Guest
room_class can be High Low
total_quantity greater than 1
room[amenities_attributes][name] Amenities name
room[amenities_attributes][0][description] Amenities description
room[amenities_attributes][0][is_available] Amenities available? true or false
room[room_rates_attributes][0][rate_type] can be standart rate, premium rate
room[room_rates_attributes][0][min_price] min price of room
room[room_rates_attributes][0][max_price] max price of room
room[room_rates_attributes][0][is_active] rate active or not? can be true or false
